Комментарии к исходникам.

1. Maven и JUnit не использовал специально, т.к. посчитал, что это будет нарушением правила об неиспользовании сторонних библиотек.
2. src/ - исходный java-код проекта
3. resources/ - содержит файл proxy.properties и манифест для создания запускаемого jar, поэтому при компиляции исходного кода
   содержимое папки resources должно быть скопировано в папку с байт-кодом
4. test/ - модульные тесты
5. для запуска нужно ипользовать класс odk.Runner. При запуске можно передать аргумент - путь к файлу proxy.properties
   (файл переопределит аналогичный proxy.properties из папки resources)
